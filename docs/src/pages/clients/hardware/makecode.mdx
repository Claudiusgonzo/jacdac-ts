# MakeCode Clients

The [pxt-jacdac-services](https://github.com/microsoft/pxt-jacdac-services) contains MakeCode support many JACDAC services.
The following editors are currently supported:

* Arcade beta https://arcade.makecode.com/beta
* Maker https://maker.makecode.com

## Usage

* Open one of the supported editors.
* Open or create a new project.
* Import the https://github.com/microsoft/pxt-jacdac-services package in your project.
* You should see a **JACDAC** section in the toolbox, under **Advanced**.


## Open Source

The sources of the MakeCode clients are in GitHub at https://github.com/microsoft/pxt-jacdac-services.
